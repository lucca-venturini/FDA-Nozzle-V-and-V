# FDA-Nozzle-V-and-V
OpenFOAM CFD validation against FDA Nozzle Benchmark for biomedical blood flow. Includes V&amp;V with PIV data (NRMSE &lt;5%), GCI mesh independence study (Roache r=2.0), y+ verification, and BC sensitivity analysis. Demonstrates regulatory-grade practices for medical device simulations. Re=500 laminar sudden expansion.
